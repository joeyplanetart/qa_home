# 使用文档

## 访问方式

- **本地：** http://localhost:8765（运行 `./scripts/run.sh` 后）
- **线上：** 部署在 Vercel 后的域名（见 [部署文档](DEPLOY.md)）

> 必须通过后端服务访问，不能直接用浏览器打开 `index.html` 文件。

## 项目 Tab

### 浏览项目

项目按业务线分组：

- 🏢 PlanetArt 内部（ADMIN、Devtools、Cordial 等）
- ☕ CafePress
- 🎴 卡片 & 印刷
- 📱 手机壳
- 🎁 礼品 & 生活

每张项目卡片显示：

- 项目名称、ID、域名
- 环境标签（prod / pre / staging）
- 备忘录 / 流程 / 片段数量统计
- 健康检查状态（绿 / 黄 / 红）

### 筛选与搜索

- **顶栏项目下拉：** 选中后，备忘录 / 流程 / 片段 Tab 只显示该项目的内容
- **项目 Tab 搜索框：** 按名称、域名、ID 过滤
- **分组按钮：** 快速切换业务线

### 健康检查

点击 **🩺 健康检查** 按钮，后端会探测各项目站点 HTTP 可达性：

- 🟢 正常（2xx / 3xx）
- 🟡 异常响应
- 🔴 不可达或超时

默认每 5 分钟自动刷新一次。

### 快捷操作

项目卡片底部三个按钮：

| 按钮 | 作用 |
|------|------|
| 备忘录 | 筛选该项目并跳转到备忘录 Tab |
| 流程 | 筛选该项目并跳转到操作流程 Tab |
| 访问站点 | 新窗口打开项目域名 |

## 备忘录

用于记录测试环境地址、Bug 排查要点、API 说明等。

### 创建 / 编辑

1. 点击右上角 **📝** 或点击已有卡片
2. 填写标题、内容
3. 选择所属项目（留空 = 通用/跨项目）
4. 选择分类、颜色，可勾选置顶
5. 保存

### 分类筛选

顶部分类标签：全部 / 通用 / API / 数据库 / 环境 / Bug / 其他

### 搜索

全局搜索框可搜备忘录标题和内容，点击结果直接打开编辑。

---

## 操作流程

记录步骤化的业务操作，例如：

- ADMIN 订单状态变更
- STI 新用户注册冒烟
- 生产环境发布流程

### 创建

1. 点击右上角 **⚙️**
2. 填写标题、描述
3. 步骤每行一个
4. 标签用逗号分隔（如 `deploy, production`）
5. 选择所属项目

### 查看

点击流程标题展开 / 收起步骤详情。

---

## 代码片段

存放常用 SQL、curl、Git 命令等，支持一键复制。

### 创建

1. 点击右上角 **💻**
2. 填写标题、选择语言、粘贴代码
3. 可选描述和标签
4. 选择所属项目

### 复制

鼠标悬停片段卡片，点击 **📋 复制**。

---

## 工具 Tab

管理 QA 常用工具链接（内部工具、测试辅助页面等），可按分类筛选。

创建方式：点击右上角 **🛠️**。

---

## 侧边栏

### 快捷链接

固定常用外部链接（JIRA、Confluence、TestRail 等）。

- **+ 添加：** 名称、URL、Emoji 图标
- 点击链接在新窗口打开
- 悬停显示编辑 / 删除

### 每日清单

每日 QA 例行任务 checklist。

- 点击任务切换完成状态，顶部进度条同步更新
- **↺ 全部重置：** 取消所有勾选（新一天开始）
- **+ 添加：** 新增任务项

---

## 全局搜索

顶部搜索框支持搜索：

- 项目名称、域名
- 备忘录标题与内容
- 操作流程标题、描述、步骤
- 代码片段标题、代码、描述

输入即显示下拉结果，点击跳转对应 Tab 并定位内容。

点击右上角 **🌙 / ☀️** 切换深色 / 浅色模式，偏好会自动保存。

## 数据说明

- 所有内容保存在数据库中（Turso 或本地 SQLite），刷新页面不丢失
- 项目列表配置在 `server/static/assets/js/projects.js`，修改后需刷新页面
- 删除操作无确认弹框，执行后通过右下角 toast 提示结果

## 常见问题

**Q: 页面提示「数据加载失败」？**  
A: 确认 `./scripts/run.sh` 已启动，且端口未被占用。

**Q: 项目 Tab 空白？**  
A: 硬刷新（Cmd+Shift+R），或重启服务。

**Q: 如何新增一个 PlanetArt 项目？**  
A: 编辑 `server/static/assets/js/projects.js` 中的 `PROJECTS` 数组，按现有格式添加 `id`、`short_name`、`domain`、`group` 等字段。

**Q: 通用内容和项目内容有什么区别？**  
A: 创建时可不选项目（标记为「通用」），这类内容不属于任何单一项目；选中项目筛选后只显示该项目专属内容。
